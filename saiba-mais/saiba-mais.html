Saiba mais sobre a evolução do Front-End

HTML Antigo (1990s)

  • Layout com <table>
  • Estilo com <font>
  • Centralização com <center>
  • Sem div, class, id
  • Links sempre sublinhados

HTML Moderno (2020s)

  • Layout com Flexbox e Grid
  • Estilo com CSS externo e responsivo
  • Semântica com <section>, <article>
  • Controle com class e id
  • Links estilizados com text-decoration: none

Curiosidade

Antes do CSS, desenvolvedores usavam tabelas para tudo — até para criar colunas e espaçamentos. Hoje, com apenas algumas linhas de CSS, conseguimos fazer layouts responsivos, animações e efeitos visuais que antes exigiam muito esforço.
